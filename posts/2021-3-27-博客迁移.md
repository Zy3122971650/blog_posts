## 搭建博客过程中用到的一些工具

+ Vue全家桶
+ Element-plus (优秀的UI框架)
+ Marked （用来把markdown文档解析成HTML）
+ Github  （储存原始文章、处理过的文章、网站的静态文件）
+ Github Action （用于自动化的对文章进行一些处理，我只需要写好文档然后commit就行啦！）
+ Github Pages（白嫖的部署网站，咳咳）
+ Github Action的流程徽章！好看！
